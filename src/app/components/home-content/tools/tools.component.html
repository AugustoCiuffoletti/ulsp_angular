Con semplici operazioni 'point and click' visualizza su una mappa il contenuto del database

Con operazioni 'point and click' sulla mappa si collezionano punti che possono essere salvati su un file locale. I file possono essere manipolati con QGis ed altri strumenti che consentano di importare file GeoJSON

Consente di operare sul database con operazioni semplici su gruppi di features GeoJSON contenute in file locali
